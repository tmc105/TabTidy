# TabTidy

A lightweight, open source tab manager that organizes tabs into groups and suspends them to save memory.

## Features

- **One-Click Organization**: Instantly group all open tabs into color-coded sessions
- **Auto-Suspend**: Automatically suspend inactive tabs after your chosen delay (1-60 minutes)
- **Smart Whitelist**: Prevent specific sites from being auto-suspended
- **Instant Resume**: Suspended tabs reload automatically when you switch back to them
- **Keyboard Shortcuts**: Quick access with Ctrl+Shift+Y (customizable)
- **Lightweight**: Minimal memory footprint and clean interface

## Installation

Install TabTidy from the [Chrome Web Store](#) *(link coming soon)*

## Usage

### Grouping Tabs
- Click the TabTidy icon in your toolbar to instantly group all open tabs
- Tabs are organized into color-coded "Session" groups
- Already-grouped tabs remain untouched

### Auto-Suspend
1. Right-click the TabTidy icon ‚Üí **Options**
2. Choose your auto-suspend delay (or disable it)
3. Inactive tabs will automatically suspend to save memory

### Whitelist Sites
- Right-click on any page ‚Üí **TabTidy: Whitelist this Domain**
- Whitelisted sites never get suspended
- Manage your whitelist in Options

## Keyboard Shortcuts

- **Ctrl+Shift+Y** (Windows/Linux) or **Cmd+Shift+Y** (Mac): Tidy tabs
- Customize in Chrome settings: `chrome://extensions/shortcuts`
